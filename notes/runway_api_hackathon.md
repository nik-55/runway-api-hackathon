How it Works
You get a weekend, API credits and direct access to the engineers who built the API. We'll go live Friday morning with a showcase of what Runway can do and a technical walkthrough so you're not starting cold. Build whatever you want — solo or with a team — and submit your project with a demo video by Monday, May 11th at 9am ET. No pitch decks. No approval chains. Just build something and show us what it does.

What to build
The API Hackathon is about building agents and applications that create, manipulate or orchestrate media using AI. Think beyond single API calls — we want to see systems that chain together creative decisions, respond to inputs and produce something people actually want to use.

A few directions to spark ideas:

An autonomous agent that generates and iterates on video content based on a brief
A tool that lets non-technical users direct AI video production through natural language
A workflow that uses Runway's API to automate a real media production pipeline
An interactive app built on the Runway Characters API
These are suggestions, not constraints. The only rule is that your project runs on Runway's API.

The Criteria
